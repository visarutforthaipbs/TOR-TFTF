# Project Overview: ทันฝุ่น ทันไฟ (TunFoon TunFai)

## 1. Background & Rationale
The PM 2.5 fine particulate matter crisis is a severe issue impacting Northern Thailand, particularly Chiang Mai. Solving this requires proactive communication to build awareness and public participation. The "ทันฝุ่น ทันไฟ" project aims to develop proactive communication strategies and build an "Active Citizen" network by utilizing Big Data. This platform serves as a central tool to gather, analyze, and present data, as well as showcase communication outputs and knowledge generated by the project in a systematic and accessible way.

## 2. Objectives
*   **Data Dashboard:** Develop a dashboard to present crucial PM 2.5 data in an easy-to-understand, modern format tailored to various stakeholders.
*   **Content Platform:** Create a central platform to publish project media, including articles, infographics, video clips, documentaries, and Long-form Interactive Journalism.
*   **Active Citizen Network:** Support the creation of a citizen network by providing access to data and a channel for local situational reporting (C-Site).
*   **Project Tracking:** Serve as a channel to monitor and evaluate the project's communication and public participation efforts.

## 3. Target Audience
*   Research teams and project staff.
*   Government agencies, private sectors, and NGOs involved in PM 2.5 mitigation.
*   Active Citizen networks and the general public interested in PM 2.5 data.
*   Mass media and journalists.
*   Academics and researchers.

## 4. Core Features (Alpha/Beta Phase)
### 4.1 Dashboard (ฝุ่นไฟใกล้ฉัน)
*   **Interactive Map:** Real-time PM 2.5 sensor data and hotspot heatmaps.
*   **Data Visualization:** 24-hour historical PM 2.5 trends using bar charts and data tables.
*   **Health Advisory:** Color-coded health impact warnings based on current AQI levels.

### 4.2 Content Hub (แพลตฟอร์มสื่อสารและความรู้)
*   **Long-form Interactive Journalism:** In-depth articles supporting multimedia and interactive elements.
*   **C-Site (Citizen Reporting):** A platform for citizens to report local news and fire/dust situations.
*   **Knowledge Hub:** A repository of research, articles, and educational materials.
*   **Creative Media:** Short/long videos, infographics, and documentaries.

### 4.3 Admin Panel
*   Content management system for articles.
*   Approval workflow for citizen reports (C-Site).
*   User management and system overview.
