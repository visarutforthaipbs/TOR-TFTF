## 1. แนะนำแพลตฟอร์ม

**"ทันฝุ่น ทันไฟ"** เป็นแพลตฟอร์มออนไลน์ของ Thai PBS สำหรับติดตามสถานการณ์ฝุ่น PM2.5 และไฟป่าในประเทศไทย โดยรวบรวมข้อมูลจากหลายแหล่ง:
- **GISTDA** — ข้อมูล PM2.5 รายจังหวัด Real-time + พยากรณ์รายชั่วโมง
- **NASA FIRMS** — จุดความร้อน (Hotspot) จากดาวเทียม VIIRS (SNPP + NOAA-20)
- **Thai PBS** — ข่าว สารคดี วิดีโอสั้น บทความเกี่ยวกับฝุ่นควัน

### ความต้องการของระบบ
- Browser ที่รองรับ: Chrome, Safari, Firefox, Edge (เวอร์ชันล่าสุด)
- ใช้งานได้ทั้ง Desktop, Tablet, และ Mobile

## 3. แดชบอร์ดฝุ่นไฟ

เส้นทาง: `/dashboard`

### 3.1 แผนที่ PM2.5 รายจังหวัด
แผนที่แสดงข้อมูล PM2.5 ของ 77 จังหวัดทั่วประเทศ อัปเดตจาก GISTDA ทุก 1 ชั่วโมง

#### สีบนแผนที่ (มาตรฐานไทย)
| ระดับ | ค่า PM2.5 (µg/m³) | สี | ความหมาย |
|:------|:-------------------|:---|:---------|
| ดีมาก | 0 – 25 | 🟢 เขียว | ปลอดภัย |
| ดี | 26 – 37.5 | 🟡 เหลือง | รับได้ |
| ปานกลาง | 38 – 75 | 🟠 ส้ม | เริ่มมีผลกระทบ |
| มีผลต่อสุขภาพ | 76 – 100 | 🔴 แดง | อันตราย |
| อันตราย | มากกว่า 100 | 🟣 ม่วง | อันตรายมาก |

#### วิธีใช้แผนที่
- **Zoom** — ใช้ Scroll wheel หรือ ปุ่ม +/- บนแผนที่
- **Hover บนจังหวัด** — แสดง Tooltip: ชื่อจังหวัด (ไทย/อังกฤษ), ค่า PM2.5, ระดับ, การเปลี่ยนแปลง, ค่าเฉลี่ย 24 ชม.
- **คลิกบนจังหวัด** — Highlight จังหวัดนั้นและแสดงรายละเอียดเพิ่มเติม
- **คลิกพื้นที่ว่าง** — ยกเลิกการเลือก

### 3.2 Sidebar รายจังหวัด
ด้านขวา (Desktop) หรือกดปุ่ม (Mobile) จะแสดงรายชื่อ 77 จังหวัด:
- เรียงค่า PM2.5 **สูงสุดไปต่ำสุด**
- จุดสีแสดงระดับด้านหน้าชื่อ
- จำนวนจุดความร้อนวันนี้
- วัน-เวลาอัปเดตล่าสุด
- Legend แสดงจำนวนจังหวัดในแต่ละระดับ (ดี/เขียว: X จว., เตือน/เหลือง: X จว. เป็นต้น)
- คลิกชื่อจังหวัด → Zoom ไปยังจังหวัดนั้นบนแผนที่

### 3.3 จุดความร้อน (Hotspot)
จุดไฟ 🔥 บนแผนที่คือตำแหน่งที่ดาวเทียม VIIRS ตรวจพบการเผาไหม้ในช่วง 2 วันที่ผ่านมา:
- **ขนาดไอคอนไฟ** — ปรับตามค่า FRP (Fire Radiative Power) ยิ่งใหญ่ = ไฟรุนแรง
- **สีไอคอน** — เหลือง (FRP ต่ำ), ส้ม (ปานกลาง), แดง (สูง)
- **ความโปร่งแสง** — ตามระดับความเชื่อมั่น (high/nominal/low)
- **Hover** — แสดง FRP (MW), ระดับความเชื่อมั่น, วันที่, กลางวัน☀️/กลางคืน🌙

### 3.4 Layer Toggle
มุมขวาบนของแผนที่มีแผง Layer Control:
- ✅ **PM2.5 รายจังหวัด** — เปิด/ปิดการแสดงสีจังหวัด
- ✅ **จุดความร้อน VIIRS** — เปิด/ปิดการแสดงจุดไฟ
- สามารถตั้งเป็นแบบใดก็ได้ หรือเปิดทั้งสองพร้อมกัน

### 3.5 แผงข้อมูลผลกระทบ PM2.5
ปุ่มด้านล่างแผนที่เปิดแผงแสดง "จำนวนวันที่ PM2.5 เกิน 37.5 µg/m³" เปรียบเทียบปี 2567 กับ 2568:
- **กราฟแท่ง** — เปรียบเทียบ 2 ปี ต่อจังหวัด
- **ค้นหา** — พิมพ์ชื่อจังหวัดเพื่อค้นหา
- **กรอง** — 4 ระดับ: วิกฤต (Critical), รุนแรง (Severe), ปานกลาง (Moderate), ปลอดภัย (Safe)
- **ขยายรายละเอียด** — คลิกจังหวัดเพื่อดูข้อมูลผลกระทบเชิงลึก

## 4. พยากรณ์คุณภาพอากาศ

เส้นทาง: `/forecast`

### 4.1 เลือกจังหวัด
- แสดง 77 จังหวัดเป็น Grid (ค่าเริ่มต้น: เชียงใหม่)
- **Tab ภูมิภาค**: ทั้งหมด, เหนือ, อีสาน, กลาง, ตะวันออก, ตะวันตก, ใต้
- **ค้นหา**: พิมพ์ชื่อไทยหรืออังกฤษ
- คลิกจังหวัดเพื่อดูพยากรณ์

### 4.2 กราฟพยากรณ์
หลังเลือกจังหวัด จะแสดง:
1. **ค่า PM2.5 ปัจจุบัน** — ตัวเลขขนาดใหญ่ + Badge สีระดับคุณภาพ
2. **กราฟเส้น (Line Chart)** — พยากรณ์ PM2.5 รายชั่วโมง:
   - เส้นอ้างอิง 4 เส้น (25, 37.5, 75, 100 µg/m³)
   - สีพื้นหลัง Band ตามระดับ
   - จุดเคลื่อนไหว (Pulse) ระบุค่าปัจจุบัน
3. **Forecast Pills** — แถวของแคปซูลรายชั่วโมง:
   - เวลา + ค่า PM2.5
   - สีพื้นหลังตามระดับ
   - จุดปัจจุบันมีกรอบเน้น
   - เลื่อนซ้าย-ขวาได้

### 4.3 Tooltip
เลื่อนเมาส์บนกราฟจะแสดง:
- เวลา (HH:MM น.)
- ค่า PM2.5 (µg/m³)
- Badge ระดับคุณภาพ
- ถ้าเป็นจุดปัจจุบัน → "(ปัจจุบัน)"

## 5. คลังสื่อสารและความรู้

เส้นทาง: `/contents`

### 5.1 ภาพรวม
หน้ารวบรวมข่าว สารคดี วิดีโอ และบทความจาก Thai PBS เกี่ยวกับปัญหาฝุ่นควันและไฟป่า

### 5.2 กรองเนื้อหา
- **ประเภทสื่อ** (Media Type Tabs):
  - ทั้งหมด
  - สารคดี (Film)
  - วีดีโอสั้น (Video)
  - รายการทีวี (Tv)
  - บทความ (BookOpen)
  - สำรวจความเห็น (MessageSquare)
- **หมวดหมู่** (Category Chips):
  - แสดง 5 หมวดแมดแรก + ปุ่ม "+N หมวดหมู่" เพื่อขยายดูทั้งหมด
  - คลิกเลือก/ยกเลิกเลือกหมวด

### 5.3 ค้นหา
ช่องค้นหาด้านบน → ค้นจากชื่อเรื่อง, หมวดหมู่, ประเภทสื่อ

### 5.4 การ์ดเนื้อหา
- รูปภาพ + ชื่อเรื่อง + Badge ประเภทสื่อ
- คลิก → เปิดเนื้อหาบนเว็บ Thai PBS ในแท็บใหม่
- Grid responsive: 1 คอลัมน์ (Mobile), 2 คอลัมน์ (Tablet), 3 คอลัมน์ (Desktop)

### 5.5 ไม่พบเนื้อหา
ถ้าค้นหาหรือกรองแล้วไม่พบผลลัพธ์ จะแสดงข้อความ "ไม่พบเนื้อหาที่ค้นหา" พร้อมคำแนะนำ

## 7. การเข้าสู่ระบบ

เส้นทาง: `/login`

### 7.1 วิธีเข้าสู่ระบบ
1. คลิกปุ่ม **"เข้าสู่ระบบ"** บนมุมขวาบนของเว็บไซต์
2. คลิกปุ่ม **"เข้าสู่ระบบด้วย Google"** ที่หน้า Login
3. เลือก Google Account ในหน้าต่าง Popup
4. เข้าสู่ระบบสำเร็จ → ไปหน้าคลังข้อมูลเปิดอัตโนมัติ

### 7.2 หลังเข้าสู่ระบบ
- Header จะแสดง Avatar และชื่อของคุณแทนปุ่ม "เข้าสู่ระบบ"
- คลิก Avatar → Dropdown แสดง:
  - **คลังข้อมูล** → ไปหน้า Data Portal
  - **ออกจากระบบ**

### 7.3 ออกจากระบบ
- คลิก Avatar → **"ออกจากระบบ"**
- เว็บจะกลับสู่สถานะ Guest

## 8. คลังข้อมูลเปิด

เส้นทาง: `/data-portal` (ต้องเข้าสู่ระบบก่อน)

### 8.1 ข้อมูลโครงการ (Local Datasets)
- ชุดข้อมูลที่ทีมโครงการจัดทำ เช่น CSV ข่าว Thai PBS, ข้อมูลผลกระทบ PM2.5
- คลิกปุ่ม **"ดาวน์โหลด"** → เปิดลิงก์ดาวน์โหลดไฟล์

### 8.2 แหล่งข้อมูลภายนอก (External Datasets)
- ลิงก์ไปยัง API หรือเว็บไซต์ที่เป็นแหล่งข้อมูลดั้งเดิม เช่น GISTDA, NASA FIRMS, Open-Meteo
- คลิก → เปิดเว็บไซต์ภายนอกในแท็บใหม่

### 8.3 ข้อมูลที่แสดง
แต่ละ Card แสดง:
- ไอคอน + ชื่อชุดข้อมูล
- คำอธิบาย
- แหล่งที่มา
- Badge รูปแบบ (CSV, API, Website)

## คำถามที่พบบ่อย (FAQ)

### Q: ข้อมูล PM2.5 อัปเดตบ่อยแค่ไหน?
A: ข้อมูลอัปเดตจาก GISTDA ทุก 1 ชั่วโมง

### Q: จุดความร้อนแสดงข้อมูลย้อนหลังกี่วัน?
A: 2 วัน (วันนี้และเมื่อวาน) จากดาวเทียม VIIRS (NASA FIRMS)

### Q: ต้อง Login เพื่อดูแดชบอร์ดหรือพยากรณ์ไหม?
A: ไม่ต้อง — หน้า Dashboard, Forecast, Contents เป็น Public ดูได้โดยไม่ต้อง Login

### Q: Login แล้วได้อะไรเพิ่ม?
A: เข้าถึงคลังข้อมูลเปิด (Data Portal) สำหรับดาวน์โหลดชุดข้อมูล

### Q: พยากรณ์ฝุ่นล่วงหน้าได้กี่ชั่วโมง?
A: ขึ้นกับข้อมูลจาก GISTDA โดยปกติพยากรณ์ล่วงหน้าหลายชั่วโมง แสดงเป็นกราฟรายชั่วโมง

### Q: รองรับภาษาอะไร?
A: ภาษาไทยเป็นหลัก Interface และเนื้อหาทั้งหมดเป็นภาษาไทย
